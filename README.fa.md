# PersianOSX

> یک توزیع مدرن لینوکس که از پایه ساخته می‌شود؛ با هدف یادگیری، سفارشی‌سازی و توسعه.

* 🇮🇷 فارسی
* 🇺🇸 English: [README.md](README.md)

---

## درباره پروژه

PersianOSX یک پروژه متن‌باز برای ساخت یک توزیع لینوکس از پایه است که بر اساس Linux From Scratch (LFS) توسعه می‌یابد.

هدف این پروژه تنها ساخت یک توزیع لینوکس نیست؛ بلکه مستندسازی تمام مراحل توسعه، تصمیم‌های فنی و فرایند ساخت سیستم است تا علاقه‌مندان بتوانند مسیر طراحی و پیاده‌سازی یک سیستم‌عامل مدرن را دنبال کنند.

---

## اهداف پروژه

* ساخت یک توزیع لینوکس از سورس.
* ایجاد سیستم ساخت (Build System) شفاف و قابل تکرار.
* طراحی معماری ماژولار و قابل توسعه.
* تولید مستندات فنی کامل هم‌زمان با توسعه.
* ایجاد یک سیستم‌عامل سبک، پایدار و قدرتمند.
* فراهم کردن یک پروژه آموزشی برای علاقه‌مندان لینوکس و توسعه سیستم‌عامل.

---

## وضعیت فعلی
