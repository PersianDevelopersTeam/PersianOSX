## About

PersianOSX is an open-source Linux distribution project built from the ground up using Linux From Scratch (LFS) as its foundation.

The goal of this project is not only to create a Linux distribution, but also to document every stage of its development so others can learn how a modern operating system is designed and built.

---

## Project Goals

* Build a complete Linux distribution from source.
* Develop a reproducible and transparent build system.
* Create a clean and modular architecture.
* Provide comprehensive technical documentation.
* Produce a lightweight yet powerful operating system.
* Build an educational project for Linux enthusiasts and developers.

---

## Current Status

**Development Stage:** Pre-Alpha

The project is currently focused on designing its architecture, development workflow, documentation structure, and build environment.

## Project Philosophy

* Learn by building.
